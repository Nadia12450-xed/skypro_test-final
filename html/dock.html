Docker

Основные команды

  • Run - запуск контейнера на основе указанного образа
  • Ps - показать список активных контейнеров
  • Ps -a - показать все контейнеры, включая остановленные
  • Stop - остановить контейнер
  • Rm - удалить контейнер
  • Images - показать список всех локальных образов
  • Pull - загрузить образ из Docker Hub
  • Rmi - удалить локальный образ

Создание и работа с образами

  • Build -t - сборка образа на основе Dockerfile
  • Tag - пометить образ новым тегом
  • Push - отправка образа в Docker Hub или другой реестр

Сети и порты:

  • Network ls - показать список сетей
  • Run-p - определить соответствие портов при запуске контейнера
